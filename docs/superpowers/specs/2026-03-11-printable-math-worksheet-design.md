# 20以内加减法打印作业 - 设计规格

## 项目概述
- **类型**: 单页HTML打印作业
- **功能**: 生成可打印的20以内加减法练习题
- **目标用户**: 小学生

## 页面规格

### 页面设置
- A4尺寸 (210mm × 297mm)
- 横向布局
- 边距: 上20mm, 下15mm, 左右15mm
- 字体: 宋体/系统默认衬线体，清晰易读
- 字号: 题目14pt，标题18pt

### 页面结构
1. **标题区** (顶部)
   - 标题: "20以内加减法练习题"
   - 班级/姓名: _______    日期: _______
   - 得分: _______ / 100

2. **题目区**
   - 50道题，5列 × 10行
   - 格式: `□ + □ = ____` 或 `□ - □ = ____`
   - 题号在左侧

3. **页脚**
   - 页码 (如需要多页)

### 题目生成规则
- 答案范围: 10-20
- 加法: 两个加数在1-19之间，和为10-20
- 减法: 被减数在11-20之间，减数在1-9之间，差为10-20
- 随机生成，每页约25道加法 + 25道减法

### 打印优化
- 黑白打印友好
- 不需要背景色
- 线条清晰
- 避免彩色内容
